# QuadraILearn Mobile Privacy Policy

Last updated: March 20, 2026

## Overview

QuadraILearn collects and processes account, learning, and device-related information required to operate the mobile app and connected web platform.

## Data We Collect

- account details such as name, email address, phone number, and role
- authentication information required for sign-in and session management
- student profile data such as class, board, school, target exam, and uploaded profile image
- usage data related to diagnostics, reports, weak-topic reviews, tokens, referrals, and admin actions
- guardian and admin management data where relevant to the account role

## How We Use Data

We use data to:

- create and secure user accounts
- authenticate users and manage sessions
- generate diagnostics, reports, and learning recommendations
- process referral and token activity
- support guardian and admin workflows
- send verification, password reset, and service-related emails

## Data Sharing

We do not sell personal information. Data may be processed by service providers required for hosting, authentication, storage, email delivery, analytics, or support operations.

## Security

We apply reasonable administrative, technical, and organizational safeguards. On supported native devices, mobile session data is stored using secure device storage.

## User Rights

Users may request account access, correction, or deletion by contacting support.

## Account Deletion

See `docs/account-deletion.md` for deletion handling and support flow.
