# QuadraILearn Mobile

Expo-based mobile app for student and guardian users against the live backend.

## Setup

1. `cd mobile`
2. `npm install`
3. Create `.env` from `.env.example`
4. `npm run start`

## Current scope

- Email/password auth against the live backend
- Student app:
  - dashboard
  - study plan
  - leaderboard
  - profile
  - diagnostic subject entry
- Guardian app:
  - overview
  - linked students
  - invite student
  - create student
